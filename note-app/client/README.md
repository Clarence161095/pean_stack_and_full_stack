# Note App

## Setup

### Tailwind CSS

- https://tailwindcss.com/docs/guides/vite

### Sass

```bash
npm install sass --save-dev
```

### Text Editor

```bash
npm install draft-js draftjs-to-html react-draft-wysiwyg
```

### Lodash

```bash
npm i lodash
```

### Firebase

```bash
 npm install firebase
```

### React Router

```bash
npm install react-router-dom
```
